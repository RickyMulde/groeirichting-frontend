# Brochures Map

Plaats hier je PDF brochures.

## Bestandsnaam
- **Bestandsnaam voor upload:** `GroeiRichting-Brochure.pdf`
- **Link in code:** `/brochures/groeirichting-brochure.pdf`

**Let op:** De bestandsnaam in de link is in kleine letters. Zorg ervoor dat de bestandsnaam op de server ook in kleine letters is, of pas de link aan in `Home.jsx`.

## Gebruik
De brochure wordt gedownload wanneer bezoekers op de "Download brochure" button klikken op de homepage.
